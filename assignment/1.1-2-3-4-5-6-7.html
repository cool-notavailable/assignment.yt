2.HTML Entities

An HTML entity is a piece of text ("string") that begins with an ampersand (&) and ends with a semicolon (;). Entities are frequently used to display reserved characters (which would otherwise be interpreted as HTML code), and invisible characters (like non-breaking spaces)

5.Tab index

The tabindex attribute is used in HTML to specify the order in which elements should receive focus when the user navigates through a web page using the keyboard. It is particularly useful for improving accessibility and allowing users to easily navigate through interactive elements without relying solely on a mouse.

The tabindex attribute accepts a numeric value that represents the tab order. Elements with a lower tabindex value will be focused first when the user presses the Tab key. If multiple elements have the same tabindex value, they will be focused in the order they appear in the HTML markup.

6.Semantic tags

  1. header :Represents the introductory content or a container for a group of introductory content at the top of a page or section.
  2. nav :Defines a section containing navigation links for the website.
  3. main : Represents the main content of the document. There should be only one
    element per page.
  4. section : Defines a section within a document. It can be used to group related content together.
  5. footer :Represents the footer of a document or a section. It typically contains information about the author, copyright information, or links to related documents.
